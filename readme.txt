== Description ==

Generate qualified leads. The Easy Quote widget guides your website visitors through your service directory with a smart wizard. The wizard captures contact details and presents the customer with qualifying questions based on service requested. 

No coding required. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Go to your Plugins -> Add New page in your WordPress admin
2. Click Add New
3. Search for ‘Lead Qualify Forms’ or Click ‘Upload’
4. If uploading plugin from file, browse and select downloaded plugin zip file 
5. Click the 'Install Now' button
6. Activate the plugin through the 'Plugins' menu
8. Go to the Lead Qualify Forms in the settings menu
9. Configure the widget. This requires you to register your company service directory with Certified Ace and activate Easy Quote: www.certifiedace.com/easyquotepro

Note: Required configuration variables can be found in the WordPress Integration section of www.certifiedace.com/easyquotedashbaord after activation of Easy Quote

Configuration Variables:

* Pro Number: [Required] This number is your unique number to identify your service directory.  This is obtained once Easy Quote has been activated on www.certifiedace.com/easyquotedashbaord

* Referral ID: [Required]  This is your unique referral code to provide credit for referrals to Certified Ace from your link. Learn more from www.certifiedace.com/referrals

* Button Position: [Optional] Defines the position of the Easy Quote widget button. Options are “Left” and “Right”.  This is optional and defaults to “Right”

* Button Text: [Optional] Defines the text for Easy Quote widget button.  This is optional and defaults to “Get Quote”

* Button Background Color: [Optional] Defines the background color of the Easy Quote widget button.  This is optional

* Button Size: [Optional] Defines the size of the Easy Quote widget button.  This is optional. Default size is 27px
